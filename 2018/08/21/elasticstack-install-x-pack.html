x-pack介绍

x-pack是elasticstack的扩展插件，将安全、报警、监视、报告和图形功能包含在一个易于安装的软件包中。通过这个插件可以监控到各个组件的运行状态，可以第一时间获取到安全信息。

配置 SSL 认证

从 elasticsearch 6.x版本开始，在更新 x-pack的证书为黄金版以上，原有的用户管理功能就必须使用 SSL 加密的通信机制方可使用。不然在导入及查看 license的时候就会报错。
